Welcome to TicketDojo!


The TicketDojo is a web application that manages and maintatins lists of technical issues, as needed by an organization. This application helps people within the organization to communicate with the IT department with regards on technical issues.

The goal of this web-application is to help the people within an organization as fast as we could by the help of creating tickets.

Tickets are created and use as the means for the communication of the two parties. The ticket mainly contains vital informations such as the name of the sender, their department, the subject of the technical issue and the detailed description of the technical issue.



How to use the app:

1.) Press the button to create a new ticket.

2.) You can view your current ticket forms by clicking on the button.

Note: Only admins can create new users for both administrators and normal users. Admins can create a new account for a user by clicking the at the top right corner
